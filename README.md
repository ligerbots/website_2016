The new [LigerBots main website](https://ligerbots.org). It uses WordPress for a backend, with a custom PHP frontend.

## Carpool Website
To test carpool website connection:
```bash
# If config is loaded
php -S localhost:8000
# If config is not loaded
php -c "<path to config>" -S localhost:8000
```
Then go to `/carpool_new_test.php`
